About

This plugin integrates Aifo.pro payment gateway into Paymenter, allowing you to accept online payments بسهولة and automatically.

Purpose

The main goal of this plugin is to enable businesses using Paymenter to accept payments from customers in Ukraine and across Europe via Aifo.pro.

Features
Accept payments through Aifo.pro
Supports customers from Ukraine and Europe
Seamless integration with Paymenter
Automatic payment processing and service activation
Use Case

If you are running a service (e.g., hosting, subscriptions, or digital products) with Paymenter, this plugin allows you to receive payments without manual handling.

Installation
Download or clone this repository
Upload the plugin to your Paymenter installation
Configure your Aifo.pro API credentials
Enable the plugin in the Paymenter admin panel
